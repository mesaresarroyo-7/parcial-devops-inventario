# Proyecto Parcial DevOps

Proyecto de automatización de inventario usando Linux, Python, Git y REST API HTTPS en la VM DEVASC.

## Funcionalidades

- Uso de comandos Linux y administración de permisos.
- Procesamiento de datos JSON, CSV y YAML.
- Consumo de API REST segura por HTTPS.
- Manejo de errores HTTP 4xx/5xx y errores de conexión.
- Versionamiento local con Git.
- Generación de reporte de inventario.

## Ejecución

```bash
python3 scripts/inventory_automation.py
cat reports/inventory_report.txt
```
